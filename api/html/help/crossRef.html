CrossReference

This function will cross reference ICAO hex codes with callsign numbers and vice versa

API Arguments

This function takes either icao_id or callsign, but not both.

  • "icao_id": Aircraft ICAO identifier, hex code
  • "callsign": Aircraft callsign.

Example query: /crossRef?icao_id=C06359 (Must be a GET HTTP request)

Example query: /crossRef?callsign=C-FFGH (Must be a GET HTTP request)
