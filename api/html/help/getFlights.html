Get List of flights from a specific aircraft

This function will get a list of flights of a given aircraft, limited to max_results results.

API Arguments

  • "icao_id" (Required): Aircraft ICAO identifier, 21 bit hex code
  • "max_results": Maximum number of flights to retrieve, 0 for infinite, not required.

Example query: /getFlights?id=C06359 (Must be a GET HTTP request)
