Get List of recently seen flights

This function will get a list of flights from a given time period, and a given station.

API Arguments

  • "max_time" (Required): Minutes in the past to view, must be integer multiple
  • "max_results": Maximum number of flights to retrieve, 0 for infinite, not required.
  • "station_id": ID number of station to look at, do not include for all stations.

Example query: /getRecentFlights/?max_time=10 (Must be a GET HTTP request)
