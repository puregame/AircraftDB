Set data for a Stations

This function will set a new station ID with a given description and position

API Arguments

  • "station_id" (required): Unique identifier of aircraft, also known as ICAO hex code
  • "description" (required): description of this aircraft
  • "lat" (required): lattitude of station
  • "lon" (required): longitude of station

Example query: /setStations/?station_id=2&description=waterloo&lat=43.45&lon=-80.51 (Must be a POST HTTP request)
