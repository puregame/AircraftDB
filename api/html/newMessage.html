New Message function

This is the new message function of the database API. This function will insert data into the database. The request should be an HTTP POST with a set of arguments with specific user data.

API Arguments

  • "id" (required): Unique identifier of aircraft, also known as ICAO hex code
  • "signal" (required): signal strength of message
  • "time" (required): timestamp of when message was recieved (timestamp with time zone) "YYYY-MM-DD HH:MM:SS.000000-TZ"
  • "station" (required): unique identifier of station which is sending data (integer)
  • "sqk": Squawk code of aircraft (integer)
  • "mode": Transponder mode (text)
  • "lat": Lattitude of aircraft (double precision)
  • "lon": Longitude of aircraft (double precision)
  • "speed": Speed of aircraft(smallint)
  • "heading": Heading of aircraft (integer)
  • "flight": Flight number or aircraft callsign (text)
  • "altitude: altitude of aircraft (integer)

Example query: /newMessage/id=C0635F&flight=ACA150&altitude=0&speed=0&heading=0&signal=7&mode=a&lat=0.000000&lon=0.000000&station=1&time=2015-10-17+20:14:2.000000-05
